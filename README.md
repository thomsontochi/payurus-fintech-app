# Payurus Fintech App

Payurus™ is a mobile-first digital wallet and neobank experience that combines local and cross-border transfers, card management, and payment workflows in a PWA-ready Vue 3 application.

## Key Features

- **Dashboard Overview** – Account cards, balance insights, cashback promos, and actionable notifications.
- **Transfer Flows** – Send, request, international, secure, and scheduled transfers with tailored UI shells.
- **Payments Hub** – Deposit options, bank transfer selection, and QR-driven actions for quick pay/receive journeys.
- **Multi-Language Ready** – Vue I18n support with structured locales.
- **Offline-Friendly** – Service worker registration, remote config caching, and PWA manifest configuration.

## Tech Stack

- Vue 3 (Composition API) & Vite
- TypeScript with Vue TSC type checks
- Pinia for state management
- Vue Router & Vue I18n
- Tailwind CSS for styling
- Vitest & Vue Test Utils for unit testing

## Getting Started

```sh
npm install
npm run dev
```

## Scripts

- `npm run build` – Type-check, compile, and bundle for production.
- `npm run preview` – Preview the production build locally.
- `npm run test:unit` – Execute unit tests with Vitest.
- `npm run lint` – Lint and auto-fix with ESLint.
